# Motivational Quotes Generator

A beautiful, responsive web app that displays random motivational quotes with a clean, modern interface.

## Features

- 🎨 Modern, responsive design using Tailwind CSS
- 📱 Mobile-friendly interface
- 🔄 Random quote generation
- 📋 Copy quotes to clipboard
- ⚡ Fast loading with smooth animations
- 🌐 CORS-free API integration

## Live Demo

Deployed on GitHub Pages: [View Live Demo](https://[your-username].github.io/quote-generator/)

## Technologies Used

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework
- **Vanilla JavaScript** - No framework dependencies
- **Quotable.io API** - Free quotes API

## Getting Started

### Local Development

1. Clone this repository:
```bash
git clone https://github.com/[your-username]/quote-generator.git
cd quote-generator
```

2. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Then visit http://localhost:8000
```

### Deployment

This project is designed to work perfectly with GitHub Pages:

1. Push to your GitHub repository
2. Enable GitHub Pages in repository settings
3. Select source branch (usually `main` or `gh-pages`)
4. Your site will be live at `https://[your-username].github.io/quote-generator/`

## API Integration

The app uses the [Quotable.io API](https://github.com/lukePeavey/quotable) which provides:
- ✅ CORS-enabled requests
- ✅ Reliable service
- ✅ High-quality quotes
- ✅ No API key required
